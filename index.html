Search-My-City

PROJECT DESCRIPTION

SearchMyCity is a website similar to the yellow pages that was popular half a decade ago. It is an effort to list all the businesses or any kind of location which are not much popular to get listed on services like google maps etc. but still people tend to search for them as these places can be needful to only a specific locality.
So in order for the people to save their time, instead of wandering they can find the place of their need on SearchMyCity.
Any individual or organisation can post to thier place and related information so that the needy can access it when needed.
It can work as a mechanism of free advertisement as well as to generate awareness to the people new to a locality about the various locations that can be useful to them.
